  A lightweight library for computing Kernel Stein Discrepancy (KSD) with JAX support
</p>

## Install

### Pypi

```{bash}
pip install ksd-metric
```

### Build from Source

```{bash}
# Install uv first
uv python install 3.10
uv sync
uv pip install .
```
